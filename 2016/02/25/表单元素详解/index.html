表单元素

表单元素详解

<input type="" value="" name=""/> 

输入框 text password

当type="text"时，输入框为文本输入框 

当type="password"时, 输入框为密码输入框 

name 为文本框命名，以备后台程序ASP 、PHP使用

value 为文本输入框设置默认值。(一般起到提示作用) 

<input type="text" name="XX.php" value="我的type是text"/>
<input type="password" name="XX.php" value="password"/>

单选框 复选框 radio checkbox

当type="radio"时，控件为单选框

当type="checkbox"时，控件为复选框

value 提交数据到服务器的值（后台程序PHP使用）

name 为控件命名，以备后台程序ASP、PHP使用

<label></label> <!--为 input 元素定义标注（标记）

label元素不会向用户呈现任何特殊效果。不过，它为鼠标用户改进了可用性。如果您在 label 元素内点击文本，就会触发此控件。就是说，当用户选择该标签时，浏览器就会自动将焦点转到和标签相关的表单控件上

标签的 for 属性应当与相关元素的 id 属性相同，for属性可把 label 绑定到另外一个元素。请把 "for" 属性的值设置为 相关元素的 id 属性的值

例如：<label for="male">Male</label>
